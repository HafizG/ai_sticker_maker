🎨 Sticker CDN

Static CDN for AI Sticker Maker app

Served via GitHub Pages

API Endpoints

GET /index.json
App bootstrap — categories, regions, zones, version
GET /regions/{CODE}.json
Region pack list — e.g. /regions/PK.json
GET /regions/{_zone}.json
Zone fallback — e.g. /regions/_arab.json
GET /regions/_default.json
Global fallback for unmatched countries
GET /packs/{id}/tray_icon.webp
Pack tray icon (96x96)
GET /packs/{id}/{n}.webp
Sticker image (512x512, n = 1..30)
